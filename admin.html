問題は自動生成されるため、編集項目はありません。

  • 🟢 初級: x² = 完全平方数 → ±整数 (例: x²=9 → ±3)

  • 練習モード / タイムアタック(全5問、目標タイム内でボーナスピース)

因数分解ディフェンス は、迫ってくる式を因数分解で撃ち落とすサバイバルゲームです。

問題は自動生成されるため、編集項目はありません。

  • 式カードがだんだん大きくなって迫ってくる → 画面いっぱいでゲームオーバー
  • x²+5x+6 = (x □)(x □) の穴埋めで撃破 ─ 符号も入力（例: +2、-3。順不同OK・x²−b²型も出題）
  • 平方型は (x □)² の1穴テンプレートで出題
  • ミスすると式が 8% 迫る / ヒントは「たして◯、かけて◯」方式
  • スピード: 🟢 初級 45秒 / 🟡 中級 20秒 / 🔴 上級 13秒 / 🟣 最上級 40秒（撃破ごとに少し加速）
  • 🟣 最上級: x²の係数が1でない高校範囲（例: 2x²+7x+3 = (2x+1)(x+3)、(□x □)(□x □) の4穴）
  • 🟢 初級のみ段階ヒント: 35%接近で「たして◯かけて◯」→ 65%接近で片方の答えを表示
  • 1ゲーム = プールから毎回ランダムに10問（初級〜上級は共通31問、最上級は15問プール）
  • 全10問クリアでクリアボーナス: 🟢3 / 🟡5 / 🔴10 / 🟣20 グリンピース（途中でゲームオーバーだと0）
  • 撃破数のベスト記録(端末ごと・最大10)
